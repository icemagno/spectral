Eis o passo-a-passo:

sudo -i

cd /home/sagitarii/
wget http://mathchem.iam.upr.si/downloads/mathchem-1.1.3.spkg
wget http://sagemath.c3sl.ufpr.br/linux/64bit/Ubuntu_14.04_LTS_sage-6.8-x86_64-Linux.tar.lrz
apt-get install lrzip
apt-get remove sagemath-upstream-binary
lrzuntar Ubuntu_14.04_LTS_sage-6.8-x86_64-Linux.tar.lrz 
ln -s /home/sagitarii/sage-6.8-x86_64-Linux/sage /usr/local/bin/sage
sage -f mathchem-1.1.3.spkg
chown -R sagitarii:sagitarii sage-6.8-x86_64-Linux/

exit ( sair do SuDo )

cd /home/sagitarii/cd sage-6.8-x86_64-Linux/
make

Após instalar o SAGE, é possível executar o script geni.py. Um exemplo de execução desse script é o seguinte:

sage -c 'load("geni.py");geni("teste.txt","-a -b -c -d")'

Quando for montar o comando acima a partir do Sagitarii:
1.	troque "teste.txt" pelo nome do arquivo gerado pelo programa geng e que contém a matriz de adjacências. Como exemplo, veja o arquivo teste.txt que vai em anexo.
2.	troque "-a -b -c -d" pelas opções correspondentes aos valores que devem ser produzidos pelo script. Para saber o que significa cada opção, veja a documentação que adicionamos no cabeçalho do script geni.py.
Repare que o script geni.py invoca funções definidas em diversos scripts em Python, que também vão em anexo. Deposite todos esses arquivos em uma mesma pasta.

A saída da execução é um arquivo em forma CSV. Um exemplo desse arquivo de saída é o arquivo "saida.csv", que também vai em anexo.
