O script a invocar é "geni.py". Esse script deve ser invocado como argumento para o 
programa SAGE. Para instalar o SAGE, siga os passos a seguir:

sudo su
apt-add-repository -y ppa:aims/sagemath
apt-get update
apt-get install sagemath-upstream-binary

Após instalar o SAGE, é possível executar o script geni.py. Um exemplo de execução desse 
script é o seguinte:
sage -c 'load("geni.py");geni("teste.g6","-a -b -c -d")'

Nesta nova versão estamos rodando o script a partir do arquivo .g6 gerado pelo AWK.

Quando for montar o comando acima a partir do Sagitarii:
troque "teste.g6" pelo nome do arquivo gerado pelo programa geng e que foi particionado pelo AWK. 
Como exemplo, veja o arquivo teste.g6 que vai em anexo.
troque "-a -b -c -d" pelas opções correspondentes aos valores que devem ser produzidos pelo script. 
Para saber o que significa cada opção, veja a documentação que adicionamos no cabeçalho do script geni.py.
O arquivo de saida do geni.py chama-se "saida.csv" e será salvo no mesmo local de origem do geni.py.
O script geni.py utiliza um pacote de rotinas que deve ser instalado no ambiente do sage antes de rodar 
o geni.py. No arquivo instalar_MathChem_SAGE.rtf que está em anexo, seguem as instruções para baixar e 
instalar este script.
